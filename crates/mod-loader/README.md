# mod-loader

This crate combines the following crates into one:
- **`mod-loader-base`**
- **`mod-loader-lua`**
- **`mod-loader-runtime`**

### Features

- **`lua`** Enables the `mod-loader-lua` crate under the `mod_loader::lua` module.
- **`runtime`** Enables the `mod-loader-runtime` crate under the `mod_loader::runtime` module.
